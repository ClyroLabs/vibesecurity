# Changelog

Todas as mudanças notáveis deste projeto serão documentadas neste arquivo.

## [1.0.0] - 2026-03-03

### Adicionado

- 🔒 Ferramenta `ler_arquivo_seguro` — leitura de arquivos com proteção contra Path Traversal
- 🔍 Ferramenta `rodar_scan_trivy` — scan de vulnerabilidades, secrets e IaC com Trivy
- 🩹 Ferramenta `propor_correcao_patch` — geração de patches de correção para revisão humana
- 🛡️ Modelo Zero Trust: todas as ferramentas exigem justificativa em linguagem humana
- 📁 Sandboxing de diretório via argumento CLI
